Endpoints da API - Categoria

Abaixo estão os links para realizar operações CRUD na API de categorias da pizzaria:

  • GET - Listar Categirias: http://172.19.0.49/pizzariateste/api/v1/categoria
  • GET - Obter Categoria por ID: http://172.19.0.49/pizzariateste/api/v1/categoria/1
  • POST - Criar Categoria: http://172.19.0.49/pizzariateste/api/v1/categoria
  • PUT - Atualizar Categoria por ID: http://172.19.0.49/pizzariateste/api/v1/categoria/1
  • DELETE - Deletar Categoria por ID: http://172.19.0.49/pizzariateste/api/v1/categoria/1

Observação: Os métodos POST, PUT e DELETE devem ser testados com ferramentas apropriadas, como Postman, Insomnia ou cURL, pois não podem ser executados diretamente através de um navegador.
